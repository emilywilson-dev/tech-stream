# tech-stream

Responsive, multi-pagte website built with HTML, CSS, and JavaScript

## Features
-Fully responsive
-Mobile-first
-Responisve Navigation

## How to View

### Option 1: View live in your browser (GitHub Pages)
Once GitHub Pages is enabled (see below), open:
https://emilywilson-dev.github.io/neural-flow-email-template/

### Option 2: View locally
1. Click **Code → Download ZIP**  
2. Unzip the folder on your computer  
3. Open `index.html` in your browser to see the layout
